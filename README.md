# Transit Ranker

A web app for ranking transit stations using an Elo-style rating system. Pick your favourite station from each head-to-head matchup — ratings adjust based on the outcome and the relative ranking of the two stations, so upsetting a higher-ranked station earns more points than beating a lower-ranked one.

Currently supports Vancouver's SkyTrain network (all 54 stations across the Expo, Millennium, and Canada lines).

## How it works

Each round presents two stations with photos pulled from Wikipedia. Choose the one you prefer and their ratings update accordingly. Over time, your preferences surface as a fully ranked list.

## Built with

React, Vite, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```
